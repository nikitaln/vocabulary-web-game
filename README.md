# Игра на знание английских слов

Проект реализован на основе Spring Boot и сборщика Maven с использованием MySQL.
Приложение при нажатии кнопки "Next word" выводит каждый раз новое слово на английском
языке, ваша задача вспомнить перевод и вписать на русском.
Цель: Изучение английским слов.
